# Ministore Shopping Site

Responsive e-commerce frontend template built with HTML, CSS, Bootstrap, jQuery, and Swiper.js.

## Project Structure
Ministore/
├── css/
│   ├── bootstrap.min.css
│   ├── swiper-bundle.min.css
│   ├── spa-header.css
│   ├── spa-footer.css
│   └── style.css
├── js/
│   ├── bootstrap.bundle.min.js
│   ├── jquery-1.11.0.min.js
│   ├── modernizr.js
│   ├── plugins.js
│   ├── swiper-bundle.min.js
│   └── script.js
├── images/
├── index.html
└── README.md

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- jQuery
- Swiper.js
- Modernizr

## Features

- Responsive design
- Product sliders
- Custom navigation
- Mobile-friendly layout

## Installation

Clone the repo and open `index.html` in a browser.

```bash
git clone https://github.com/Iamfouzia/Ministore.git
```
